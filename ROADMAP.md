# Roadmap

This is a focused implementation roadmap, not a product promise.

## Near term

- Add a one-screen beginner checklist that can be printed or copied into `SETUP-NOTES.md`.
- Add an optional `first-page.html` starter template as a static example or downloadable snippet.
- Make the Setup Path Builder output separate prompts for Claude-only, Codex-only, and Claude-plus-Codex handoff workflows.
- Add a visual diff-reading explainer for beginners before they make their first commit.

## Middle term

- Add a small browser-only exercise for converting a weak chat prompt into a scoped coding-agent task.
- Add a browser-only source-note builder for Data.gov, National Archives Catalog API, and OAI-PMH.
- Add accessibility and mobile-polish review for the homepage learning-path cards.
- Expand `notebooklm/starter-bundle.md` with a short operator-facing table of contents.

## Guardrails

- Keep the project static and client-only.
- Keep beginner content practical: visible files, clear checks, no secrets, no private data.
- Prefer official current docs for fast-moving AI tooling references.
- Do not let the public site imply certification, endorsement, investment advice, or production readiness.
